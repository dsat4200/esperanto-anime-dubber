═══ ANIDUB USAGE ═══════════════════════════════


SETUP
  First run (or after recreating .venv):
    > .\install.ps1

  Installs PyTorch CUDA, ffmpeg, all dependencies, and registers CLI commands.

  If you already have the venv but need to re-register:
    > .\.venv\Scripts\Activate.ps1
    > pip install -e .


FOLDER SETUP
  Put anime in:
    anime/{name}/
      Episode.Name.S01E01.mkv
      Episode.Name.S01E02.mkv
      ...

  Each MKV must have an embedded ASS subtitle track (or an .ass file
  next to the MKV with the same stem). Japanese audio track required.


GUI WORKFLOW
  Start the editor:
    > anidub-edit

  Opens http://127.0.0.1:5000 in your browser.

  1 CREATING A PROJECT
    Enter the anime folder name (e.g. gabriel_dropout) and click Open.
    The editor discovers all MKVs, extracts audio/subtitle tracks, and
    shows each episode as a card on the home screen.

  2 OPENING AN EPISODE
    Double-click an episode card to open the editor.
      - Left sidebar shows clip list (subtitle lines)
      - Center shows a timeline + video preview area
      - Right sidebar shows the current clip's details

  3 SETTING AUDIO & SUBTITLE TRACKS
    Use the dropdowns in the top bar to select the Japanese audio track
    and the correct subtitle track.

  4 DEMUCS SEPARATION
    Click "Separate Audio" in the toolbar. This runs once per episode
    and splits the audio into vocals + background (no_vocals).

  5 TRANSLATING
    Translate All   → translates every clip in the episode via Google
                       Translate (auto-detect → Esperanto)
    Batch Translate → translates all selected episodes at once
    Per-clip        → type a manual translation in the text area or
                       click the Translate button

  6 VOICE CLONING
    Clone All       → generates Esperanto voice for every translated
                       clip in the current episode
    Batch Clone     → generates voice for all selected episodes
    Per-clip Clone  → generates voice for a single clip

  7 REVIEWING
    Click any clip in the list to hear the cloned voice alongside the
    original background audio (video preview). Adjust audio offset if
    the voice timing is off. Click Accept or Reject.

  8 ASSEMBLING
    Click "Assemble" in the top bar to build the full dubbed episode.
    Output: projects/{anime}/{episode}/out/{episode}_Dubbed.mkv

    Set "Complete" checkbox for an episode to skip it in future batches.


BATCH OPERATIONS (HOME SCREEN)

  Hold Shift + click to select multiple episode cards, then click:
    Batch Translate   Translates all selected episodes.
    Batch Clone       Generates Esperanto voice for all selected
                      episodes.

  Progress bars (Tr / Cl / Ac) on each card update live during
  batch operations. Failed episodes are reported after completion.

  Tracks can be chosen for the batch — the editor will prompt for
  audio and subtitle track selection before starting.


GPU MEMORY

  The GPU indicator in the top-right corner shows VRAM usage.
    Click it        → shows detailed panel with live tensors / reserved.
    Clear GPU Memory   → frees cached but unused VRAM pool.
    Force Unload       → drops all loaded TTS + Whisper models.

  If VRAM exceeds 75% before a clone, the shared model is
  automatically unloaded and recreated to keep within limits.
